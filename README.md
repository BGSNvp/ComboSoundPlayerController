# Combined Controller (Player + Soundboard)

Mobile-friendly controller for the BGSN combined overlay. Open on your phone to control:
- **Player card** — select players, show/hide overlay (7s auto-hide), manage roster, style fields
- **Soundboard** — trigger sound effects, manage sounds, volume control

## Tabs

- **Select** — Tap a player to select them as active
- **Roster** — Add, edit, delete players with video URLs
- **Style** — Toggle fields, font size, color, position offsets
- **Sounds** — Sound effect buttons with category filter and volume
- **Edit Sounds** — Add/edit/delete sounds, import from soundboard repo

## URLs (GitHub Pages)

- **Controller:** `https://bgsnvp.github.io/ComboSoundPlayerController/`
- **Overlay:** `https://bgsnvp.github.io/ComboSoundPlayerOverlay/`

## Setup

1. Enable GitHub Pages: Settings → Pages → Deploy from `main` branch
2. Open on your phone and start controlling!
